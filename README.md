# Introduction to scATAC Analysis

This project provides a basic introduction to single-cell ATAC-seq (scATAC-seq) analysis using the framework outlined in
[Signac tutorial](https://stuartlab.org/signac/articles/mouse_brain_vignette) and [YouTube Tutorial](https://www.youtube.com/watch?v=e2396GKFMRY)

## Data Source
The dataset used in this tutorial is taken from:  
Itokawa et al., *Nature Communications*, 2022  
[Epigenetic traits inscribed in chromatin accessibility in aged hematopoietic stem cells](https://www.nature.com/articles/s41467-022-30440-2#citeas)


## Contents
- Notebook/script following the Signac vignette workflow:
  - Data loading
  - Computing Quality control metrics
  - Data filtering
  - Clustering
  - Gene activity analysis
- References and links to the original vignette and data source
- Basic instructions to run the analysis
